About play-eventadapters

A set of event adapters for PLAY including an SDK of abstract adapters.

Project Modules

This project has declared the following modules:

Name Description
play-eventadapter-abstractrdfreceiver An abstract event adapter for receiving PLAY events, part of the adapter SDK.
play-eventadapter-abstractrdfreceiver-rest An abstract event adapter for receiving PLAY events, part of the adapter SDK.
play-eventadapter-abstractrdfsender An abstract event adapter for sending PLAY events, part of the adapter SDK.
play-eventadapter-abstractrdfsender-rest An abstract event adapter for sending PLAY events, part of the adapter SDK.
play-eventadapter-facebook A PLAY event adapter to receive real-time information from people's Facebook wall if the grant access to the adapter.
play-eventadapter-twitter A PLAY event adapter to receive real-time information the Twitter garden hose.
play-eventadapter-xively A PLAY event adapter to receive real-time information from the Xively service (formerly Cosm or Pachube).
play-eventadapter-ldstreams A PLAY event adapter to publish events in a Linked Data manner through a restful interface using HTTP 1.1 chunked connections.
